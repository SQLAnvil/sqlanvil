# sqlanvil VSCode Extension

Syntax highlighting and project compilation for sqlanvil projects in VSCode.

## Usage

This extension requires the sqlanvil CLI installed globally:

```bash
npm i -g @sqlanvil/cli
```

Includes:

- Syntax highlighting for `.sqlx` files
- Realtime compilation of your project
- `cmd + click` on a `ref()` function to navigate to the referenced file
